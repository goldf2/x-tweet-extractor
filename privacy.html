Privacy Policy

Last updated: July 5, 2026

This privacy policy describes how the X Tweet Extractor browser extension ("the Extension") handles user data. By installing and using the Extension, you agree to the practices described below.

1. Data We Collect

The Extension processes the following types of data:

  • Public tweet content — text, timestamps, and engagement metrics (likes, retweets, replies) that are publicly visible on X (Twitter) user profile pages you visit.
  • Language preference — your selected UI language, stored locally in the browser.

2. How We Use Your Data

The Extension uses the collected data solely for the following purposes:

  • To extract and display tweets from the current X profile page.
  • To export extracted tweets into downloadable files (JSON, CSV, TXT, HTML, Markdown).
  • To remember your language preference for the Extension interface.

3. Data Storage and Transmission

Local storage only. All extracted tweet data is stored temporarily in the browser's memory (page-level JavaScript variables). The Extension does NOT:

  • Upload, send, or transmit any data to any external server or third-party service.
  • Use analytics, tracking pixels, or advertising SDKs.
  • Store extracted tweet data persistently — data is cleared when you close the tab or click "Clear Data".

The only persistent storage is your UI language preference, saved via chrome.storage.local.

4. Permissions We Request

  • activeTab — Access the current X page content when you activate the Extension.
  • scripting — Inject the extraction script into x.com pages.
  • storage — Save your language preference locally.
  • downloads — Save exported files to your computer.
  • host_permissions (x.com, twitter.com) — Run only on X (Twitter) domains.

5. Third-Party Services

The Extension does NOT integrate with or send data to any third-party service, API, or analytics platform.

6. User Rights

  • You can clear all extracted data at any time by clicking the "Clear Data" button.
  • Closing the browser tab automatically discards all in-memory tweet data.
  • You can uninstall the Extension at any time to remove all stored preferences.

7. Data Security

Since no data is transmitted to external servers and all processing occurs locally in your browser, the risk of data interception is minimal. The Extension does not handle passwords, payment information, or other sensitive personal data.

8. Children's Privacy

The Extension is not directed at children under 13 and does not knowingly collect data from children.

9. Disclaimer

The Extension is intended for personal data backup and research purposes only. Users are responsible for complying with X (Twitter)'s Terms of Service and applicable laws when extracting tweet data. The Extension developers are not responsible for any misuse.

10. Changes to This Policy

We may update this privacy policy from time to time. Changes will be posted on this page with an updated revision date.
