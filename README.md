# X Tweet Extractor - Chrome 扩展

提取 X (Twitter) 用户的推文并导出为 JSON 或 CSV 格式。

## 功能

- 自动滚动提取用户主页的所有可见推文
- 提取推文文本、发布时间、点赞/转发/回复数等信息
- 支持导出为 **JSON** 或 **CSV** 格式
- 支持检测转发推文和带媒体的推文
- **支持输入用户名**：直接在输入框填写 X 用户名，自动跳转并提取
- **停止按钮**：提取过程中可随时停止，并保留已提取的数据
- 暗黑主题 UI，与 X 风格一致

## 安装方法

### 开发者模式加载（推荐）

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目的文件夹 `x-tweet-extractor`
5. 扩展图标将出现在浏览器工具栏中

## 使用方法

### 方式一：输入用户名（推荐）

1. 点击浏览器工具栏中的扩展图标
2. 在「X 用户名」输入框中填写目标用户名（不带 @，如 `elonmusk`）
3. 点击「开始提取推文」（或在输入框按回车）
4. 扩展会自动跳转到该用户主页并开始提取
5. 提取过程中可点击「停止提取」随时中断
6. 提取完成后，点击「导出 JSON」或「导出 CSV」保存数据

### 方式二：从当前页面提取

1. 打开任意 X (Twitter) 用户主页，例如 `https://x.com/username`
2. 点击浏览器工具栏中的扩展图标
3. 用户名输入框留空，直接点击「开始提取推文」
4. 扩展会自动滚动页面，收集所有可见推文
5. 提取完成后，点击「导出 JSON」或「导出 CSV」保存数据

## 提取的数据字段

| 字段 | 说明 |
|------|------|
| id | 推文 ID |
| text | 推文文本内容 |
| createdAt | 显示的时间文本 |
| timestamp | ISO 8601 格式时间 |
| likes | 点赞数 |
| retweets | 转发数 |
| replies | 回复数 |
| views | 浏览数 |
| isRetweet | 是否为转发 |
| hasMedia | 是否包含图片/视频 |
| url | 推文链接 |

## 注意事项

- 本扩展仅在 `x.com` 和 `twitter.com` 域名下工作
- 提取速度受页面加载速度影响，建议保持页面在前台
- 提取数量上限默认为 500 条推文，可在代码中调整
- X 的 DOM 结构可能会更新，如提取失败请检查浏览器控制台日志

## 权限说明

| 权限 | 用途 |
|------|------|
| activeTab | 访问当前标签页以提取推文 |
| scripting | 在页面中执行提取脚本 |
| storage | 临时存储提取的数据 |
| downloads | 导出 JSON/CSV 文件 |
